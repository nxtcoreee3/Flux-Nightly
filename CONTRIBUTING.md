# Contributing 🎮

Thanks for wanting to contribute to this game site!

# 📌 How to Add a Game

Copy this template:

```js
{
  id: 'game-id',
  title: 'Game Title',
  thumb: 'assets/game-image.png',
  url: 'https://yourname.github.io/game_name',
  desc: 'Short description of the game.'
},
```

# 🖊️ Fill in the details:

id → lowercase with dashes (example: drift-boss)

title → normal game name

thumb → image inside /assets/ (example: drift-boss.png)

url → full playable link (https://example.github.io/)

desc → short, clear description (1 sentence)

Add your game to the list in the script.js

# 🖼 Thumbnails

Use .png format

Keep names lowercase with dashes

Example: assets/drift-boss.png

# ✍️ Description Rules

Keep it short (1 sentence)

Describe the main gameplay

No emojis or extra formatting

Good example:
Drive fast, dodge traffic, and survive as long as possible.

# ✅ Rules

Make sure the game link works

No duplicate games

Games must be hosted on GitHub Pages with no advertisements

Keep formatting consistent

Keep everything clean and simple

# 🚀 Tips

Try to match existing styles

Keep IDs the same as the game name

Test your game before adding it

Thanks for contributing! 🎉
